# 2olly-Bkam
"2olly-Bkam" is a price comparison calculator of courier compatitors of Egypt.


## INSTRUCTIONS ##

Use npm package "live-server" or any equivalent server to run the application.


## TECHNOLOGIES ##

- HTML 5

- Bootstrap 3+

- AngularJs

- jQuery
